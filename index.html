Overview

If you love to do practice more and more coding problem on GeeksforGeeks then this application is for you. It helps you to save your all coding solution immediately upload it to your Github repository by clicking one button showing on your output window. you all practice the coding problem but we can't memorise all the approaches of each problem so I came with a solution that helps you to save your all approaches on your GitHub page which will help for your future purposes.

Security and Features

  • This is a page action Extention is which only works on geeksforgeeks website and internet dependable. It helps the user to upload geeksforgeeks solution to their GitHub page
  • you can upload your solution in any of your repositories by choice and before proceeding further make sure that it will be initialized with a GitHub readme file otherwise, it will show inappropriate behaviour by default it will upload all your geeksforgeeks solution your solutions to gfggit.
  • For all this request user must need to enter the GitHub personal access credentials otherwise it may not proceed further and show a user authentication error.
  • During entering your credentials make sure that any malware script is not open in browser otherwise it may create a chance for a cyber attack. I tried my best to hide and save your information from those malware scripts.
  • For all these features of extension, it needs some user permission so that it can access the user and its repository information.
  • In end, I promise that It will not share your authentication credential with anyone. It only stores the information in user browser>storage for which it can able to verify the user.
